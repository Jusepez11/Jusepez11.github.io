Project Overview

This project aims to create a comprehensive website for the university's game development club. The website will serve as a central hub for current and prospective members, showcasing club projects, highlighting member achievements, and providing information about game jams and club activities.

Intended Users

  • Current club members seeking information and resources
  • Prospective members interested in joining the club
  • University students interested in game development
  • Faculty and staff who support the club
  • Game jam participants and potential participants
  • Industry professionals or alumni interested in club achievements

Overview of Content

  • Home Page: Club introduction, latest news, and upcoming events
  • About Page: Club history, founding members, and mission statement
  • Projects Gallery: Showcase of games developed by club members
  • Game Jam Hall of Fame: Highlights of past game jam winners
  • Join Us Page: Membership information and social media links

Page Designs

Home Page (index.html)
  • Purpose: Serve as the main landing page and provide quick access to key information
  • Audience: All visitors to the website
  • Content: Hero section with club logo, news ticker, featured game project carousel, upcoming events calendar
  • Buttons/Hyperlinks: Navigation menu, social media links, call-to-action buttons
  • Actions: Carousel navigation, calendar interaction, news ticker animation
  • Special Notes: Ensure mobile responsiveness for all interactive elements
About Page (about.html)
  • Purpose: Share club history and introduce founding members
  • Audience: Prospective members, faculty, industry professionals
  • Content: Club history timeline, founding members profiles, mission statement
  • Buttons/Hyperlinks: Timeline navigation, member social links
  • Actions: Interactive timeline navigation, member profile expansion
  • Special Notes: Include high-quality photos of founding members
Projects Gallery (projects.html)
  • Purpose: Showcase games developed by club members
  • Audience: Everyone interested in the club's work
  • Content: Grid of game projects, screenshots/videos, project descriptions
  • Buttons/Hyperlinks: Play/download links, filter categories
  • Actions: Gallery filtering, lightbox for screenshots, video previews
  • Special Notes: Optimize media loading for performance
Game Jam Hall of Fame (hall-of-fame.html)
  • Purpose: Highlight past game jam winners and their projects
  • Audience: Current and prospective game jam participants
  • Content: Winners by year/event, project showcases
  • Actions: Sort by year/category, interactive game previews
Join Us Page (join.html)
  • Purpose: Provide information about joining the club
  • Audience: Prospective members
  • Buttons/Hyperlinks: Social media links, Discord invite
  • Actions: Social media integration, Discord widget
  • Special Notes: Keep sign-up process clear and straightforward

Dynamic Functionality

The website will include the following interactive features:

  • Project Showcase: Interactive gallery using jQuery lightbox plugin
  • Event Calendar: Interactive calendar showing meetings and game jams
  • Social Media Integration: Live social media feeds and Discord widget
  • Navigation and UI: Sticky navigation bar, smooth scrolling, mobile-responsive menu
