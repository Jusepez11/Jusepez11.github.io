Website CRAP Evaluation

Make Front End Shist Again

The website "Make Frontend Shit Again" is a humorous, nostalgic tribute to the early days of web development, embracing the chaotic, fun nature of retro web design. It features bold, flashy elements like animated GIFs, neon colors, and references to outdated technologies like FTP, jQuery, and Geocities. The message encourages web developers to bring back creativity and playfulness to frontend design, urging users to build "dumb" and "useless" but fun websites again.


  • Contrast: It lacks clear contrast, using flashy, chaotic colors that don't always distinguish important elements.
  • Repetition: The design lacks consistent patterns, which can confuse users about visual hierarchy.
  • Alignment: Elements are randomly placed without structure, disrupting visual flow.
  • Proximity: Irrelevant elements are grouped together, making it hard to distinguish related content.

W3schools

W3Schools is a popular platform for learning web development. It offers tutorials on a wide variety of topics including HTML, CSS, JavaScript, Python, SQL, and more. Its structure is straightforward, providing examples, code editors, and exercises.


  • Contrast: W3Schools uses high contrast between text and background, typically with dark text on a light background and vice versa for headers. This makes content highly readable and important elements easy to spot.
  • Repetition: The site maintains a consistent layout across all tutorials. Repetitive use of fonts, colors, and buttons strengthens user familiarity and navigability.
  • Alignment: Everything on the page, from headings to code snippets, is well-aligned, making the site easy to follow and visually structured.
  • Proximity: Related content, such as navigation menus and tutorials, is grouped together, helping users easily find what they need. Text and images are appropriately spaced, improving readability.
